# 다항식의 연산(3)

## 문제풀이

### 예제 4) 다음을 구하시오
#### $a+b=4$, $ab=2$일 때, 다음 식의 값을 구하시오.(단, $a>b$이다)
- (1) $a-b$
- $(a-b)^2 = (a+b)^2 - 4ab$
- $(a-b)^2 = 16-8$
- $(a-b)^2 = 8$
- $a-b = \pm2\sqrt{2}$
- $a-b = 2\sqrt{2}$

- (2) $a^3+b^3$
- $a^3+b^3=(a+b)^3-3ab(a+b)$
- $a^3+b^3=(4)^3-3\times2(4)$
- $a^3+b^3=64-24$
- $a^3+b^3=40$

- (3) $a^3-b^3$
- $a^3-b^3=(a-b)^3+3ab(a-b)$
- $a^3-b^3=(2\sqrt{2})^3+3\times2(2\sqrt{2})$
- $a^3-b^3=8\times2\sqrt{2} + 6\times2\sqrt{2}$
- $a^3-b^3=16\sqrt{2} + 12\sqrt{2}$
- $a^3-b^3=28\sqrt{2}$

#### #a+b+c=4#, #ab+bc+ca=1#일 때, #a^2+b^2+c^2#의 값을 구하시오.
- $a^2+b^2+c^2 = (a+b+c)^2-2ab-2bc-2ca$
- $a^2+b^2+c^2 = (a+b+c)^2-2(ab+bc+ca)$
- $a^2+b^2+c^2 = 4^2-2/times1$
- $a^2+b^2+c^2 = 16-2$
- $a^2+b^2+c^2 = 14$

### 유제 7) a+b=3, a^3+b^3=9 일 때, 다음 식의 값을 구하시오.(단, a>b이다)
#### (1) $ab$
- $(a+b)^3 = a^3+b^3+3ab(a+b)$
- $(3)^3 = 9+3ab(3)$
- $(3)^3-9 = 3ab(3)$
- $\frac{(3)^3-9}{3}  = 3ab$
- $\frac{18}{3} = 3 ab$
- $6 = 3 ab$
- $2 = ab$
- $ab = 2$

#### (2) $a^6+b^6$
- $a^6+b^6 = (a^3+b^3)^2-2a^3b^3$
- $a^6+b^6 = 9^2 - 2(ab)^3$
- $a^6+b^6 = 81 - 2(2)^3$
- $a^6+b^6 = 81 - 2\times8$
- $a^6+b^6 = 81 - 16$
- $a^6+b^6 = 65$
